# JSONfn

Javascript (both node.js and browser) plugin to stringify / parse / clone javascript objects with

  - **Functions**
  - **RegExp**
  - **Date**

**Basic usage:**

    var str = JSONfn.stringify(obj);

    var obj = JSONfn.parse(str);
